# Spacecraft

A simple CLI to solve the Spacecraft Problem, made with [TypeScript](https://www.typescriptlang.org/)

Let's build a rocket together!

![image](./rocket.jpg)

## The Problem

![image](./problem.png)

## Requirements

* NodeJS
* Yarn

## Install Dependencies

```sh
yarn install
```

## Build the CLI bundle

```sh
yarn build
```

## Run CLI

```sh
yarn start
```

## Run tests

```sh
yarn test
```
